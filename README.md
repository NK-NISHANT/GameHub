## Features

- **4 Games:** Gem Crush, Sudoku Pro, Memory Match, Neon Snake.
- **Unified Login:** Log in once on the website; games launch automatically.
- **Cloud Database:** Real-time leaderboard using MongoDB Atlas.

## How to Run

1. **Install:** `pip install -r requirements.txt`
2. **Configure:** Rename `.env.example` to `.env` and paste your MongoDB Connection String.
3. **Run:** `python backend/app.py`
4. **Play:** Open your browser to `http://127.0.0.1:5000`
